# Onix Burguer Artesanal

Landing page institucional desenvolvida com Next.js 14, TypeScript e Tailwind CSS para a Onix Burguer Artesanal.

## Stack

- Next.js 14+ com App Router
- TypeScript
- Tailwind CSS

## Como rodar

1. Instale o Node.js 18.18+ ou 20+.
2. Instale as dependências:

```bash
npm install
```

3. Rode o ambiente local:

```bash
npm run dev
```

4. Acesse `http://localhost:3000`.

## Estrutura

- `src/app`: layout global e landing page
- `src/components`: componentes reutilizáveis da interface
- `src/data/menu.ts`: cardápio mock estruturado
- `public/images`: placeholders visuais locais

## Destaques

- Visual premium com base escura e destaque dourado
- Layout mobile-first e totalmente responsivo
- Dados de cardápio organizados por categoria
- CTAs voltados para pedido via WhatsApp
